Information:


    Created to merge all pdf files stored in 'put_your_pdf_files' and put watermark on each page of them

    Useful when having more PDF separated files(for example book chapters) or just putting watermark on pdf files

    Using Python3



Instructions: (commands for Windows might differ)


    1. Create a new environment                                 "  python -m venv venv  " 

    2. Activate it                                              "  source venv/bin/activate  "

    3. install required libraries by executing                  "  pip install -r requirements.txt  "
    in terminal

    4. launch python watermark.py through terminal 

    5. now replace all files in "put_your_pdf_files" and "put_your_watermark" by your own :) 

    6. execute watermark.py yet again
